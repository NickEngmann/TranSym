**How to run the app**
</br>

1) Download/Clone this repo </br>
2) go to "chrome://extensions/"</br>
3) Enable Developer Mode </br>
4) Select "Load Unpacked Extension" </br>
5) Locate the Folder named "extension" in Incentive and press enter </br>
